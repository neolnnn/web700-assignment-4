Welcome to my website

This website was made for Web700 class, to demonstrate the principles of HTML5 and routing with Express.

So what does this site have?

This website contains three static views (Home, About and HTMLDemo) as well as five routes that supply data from .json files containing student and course information. We can view all students, search students by course, search a single student by their student number, search students that are TAs and view all courses.
